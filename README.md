# MWC2026 Object Detection

This example was created as an interation of the official Object Detection Demo by Arduino INC. The example adds the Arduino part where the board iluminates the modulino LEDS and makes sound with the buzzer depending of the movement and object detected. 

## Expected Behaviour

* The board detects different objects in real time using Yolo or the Gestures Network
* If a cellphone is detected the Modulino LEDs turn ON in red for some seconds and the buzzer makes some sound
* If a plant is detected the Modulino LEDs turn ON in green for some seconds and the buzzer makes some sound
* If a peace sign (change network to gestures) is detected the Modulino LEDs turn ON in white for some seconds and the buzzer makes some sound


## Additional Hardware

* 1x Modulino Movement
* 1x Modulino LEDS
* 1x Modulino Buzzer

Created for the Mobile Word Congress Barcelona 2026 as a live demo.
